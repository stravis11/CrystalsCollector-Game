1. Generate a random number between 19 - 120 and display it in the random number field.

2. Generate a random number for Crystals when clicked:
    a. Generate a random number between 1 - 12 and assign to crystal1
    b. Generate a random number between 1 - 12 and assign to Crystal2
    c. Generate a random number between 1 - 12 and assign to Crystal3
    d. Generate a random number between 1 - 12 and assign to Crystal4

3. Create on click events for each crystal that add their value to the total score.

4. Create win event if total score equals random number
    a. Wins increases by 1
    b. Restart game, do not change wins/losses

5. Create loss event if total score greater than random number
    a. Losses increases by 1
    b. Restart game, do not change wins/losses
